## Installation

```bash
# yarn
$ yarn add resumon

# npm
$ npm install resumon
```

## Development

```bash
# install dependencies
$ pnpm install

# build for production
$ pnpm build

# lint project files
$ pnpm lint

# run tests
$ pnpm test
```
